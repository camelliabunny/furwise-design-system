Typography

All text uses the Inter typeface with CJK fallbacks. Tracking (letter-spacing) is set at 2% of font size throughout.

Font stack: 'Inter', 'Gen Jyuu Gothic P', '源柔ゴシック', 'Noto Sans JP', 'Noto Sans TC', -apple-system, sans-serif
